This assignment includes the following files:
- 9045_9131_9200_HikeTheDayDB_corrected.pdf: corrected 1st deliverable of the assignment.
- hikeTheDay.mwb: database model in MySQL Workbench format.
- hikeTheDay.png: database model diagram as a png file extracted from MySQL Workbench.

- users.sql: SQL script for creating the users of the database and their rights.
- query1.sql, query2.sql, ... , query12.sql : SQL scripts that include queries to the database.
